## 🚀 How to Execute

### Step 1: Install Dependencies
```bash
npm install
```
**Time**: ~2 minutes

### Step 2: Start Development Server
```bash
npm run dev
```
**Result**: Server starts at `http://localhost:5173/`

### Step 3: Open Browser
Visit `http://localhost:5173/` in your browser

### Step 4: Login
**Admin**: admin@government.in / password123  
**Employee**: employee@state.gov.in / password123

**Total Time**: ~3 minutes from start to first use
